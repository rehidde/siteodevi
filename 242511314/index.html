SKYRIM
Oyun : The Elder Scrolls V: SKYRİM Yıl : 2011 Yapımcı : Bethesda Game Studios Tür : Action RPG Oyun Motoru : Creation Engine Yönetmen : Todd Howard Besteci : Jeremy Soule EVREN : The Elder Scrolls
The Elder Scrolls V: Skyrim, The Elder Scrolls evreninin merkezi olan Nirn adlı gezegenin Tamriel kıtasında geçer. Önceki Daggerfall, Morrowind ve Oblivion oyunlarından farklı olarak bu oyun bizi Tamriel kıtasının kuzeyine, Nord ülkesi soğuk Skyrim topraklarına götürüyor.
Baldur's Gate 3
Oyun : Baldur's Gate 3 Yıl : 2023 Yapımcı : Larian Studios Tür : RPG Oyun Motoru : Divinity Engine Yönetmen : Sven Vincke Besteci : Borislav Slavov EVREN : The Forgotten Realms
Baldur's Gate 3, Dungeons and Dragons markasının mekanikleri ile The Forgotten Realms adlı yine D&D'den aşina olduğumuz bir evrenin kurgusunu birleştiren bir RPG'dir. Önceki oyunlarda olduğu gibi bu oyunda Faerun adlı kıtada geçip, yolun yine Baldur's Gate adlı şehire düşeceği bir macerayı konu alır.
